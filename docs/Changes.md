# Deviations from Design I
Several features from the 'Design I' phase were either modified or omitted in Cycle I. This document details the motivations and reasoning behind those changes to the original plan.

_This document concerns specific changes made to the design plan discussed during meetings or over messages between team members (before and after narrowing of the scope of the entire project). For documentation on general narrowing of the scope of the project, see [ScopeChanges.md](ScopeChanges.md)._

**_Disclaimer:_**
_For all of Design I and most of Implementation I, this document had not been updated. This is because the information from the meetings/messages was recorded on personal computers. Moving forward, we will update this document actively._

## Removed
#### REQ-S-3: Staff must be able to reserve a classroom with a single click [See GitLab Issue #4]
**Reasoning:** Client removed requirement from project description. \
**Current Status:** Removed from project entirely.
